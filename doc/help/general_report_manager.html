Create new report

There are 3 options available: 1. Create report from scratch; 2. Import report; 3. Import SQL script.

Create report from scratch

  • Right click on Repors root label of navigation tree;
  • Choose New Empty Report menu item;
  • Enter the name for the new report group;
  • Rename report if needed by right click menu Rename Report item;
  • Select new report item in the navigation tree. There are 3 tabs became available for the report: SQL, Lua and htt;
  • Shoose SQL tab and inser SQL script into editor window then press Save button;
  • Execute the sql script with Test button;
  • Press Create Template button;
  • Press Run for get report html.
