Create new report

There are 3 options available: 1. Create report from scratch; 2. Import report; 3. Import SQL script.

Create report from scratch

  • Using the navigation tree, mouse Right-Click on Reports label;
  • Choose the menu item: New Empty Report;
  • Enter the name for the new report group;
  • Rename reports, as required, by right-click on menu item: Rename Report;
  • Choose the tab: SQL, insert the SQL script into the editor window, check that the SQL script finished with semicolon, then press the Save button;
  • Execute the SQL script with the Test button;
  • If test passed the Create Template button will be available. Press it to generate html template for the report;
  • Press Run for get report html.
