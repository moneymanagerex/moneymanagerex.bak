MMEX Documentation using Doxygen
================================
This file is provided to allow future developers to obtain documentation for MMEX by using Doxygen.


Doxygen is available as follows:
===============================
Doxygen Home Page: http://www.stack.nl/~dimitri/doxygen/index.html
Doxygen  Download: http://www.stack.nl/~dimitri/doxygen/download.html


Supplied File:
==============
doxyfile.dox     - Used by the Doxywizard to generate the documentation from the source code.


Doxygen Actions:
================
Create a sub directory (folder)  : ./html
Generated documentation is add to: ./html
Allow documentation to be viewed.

Generated documentation can also be viewed with a web browser using:
./html/index.html
